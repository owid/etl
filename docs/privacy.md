# Privacy notice

This documentation site is maintained by [Our World in Data](https://ourworldindata.org/).

## What we collect

Nothing. This site does not collect or store any personal data, set cookies, or run analytics.

## Third-party resources

Some pages load external resources to render content correctly. When this happens, your browser makes requests to these services, which may briefly see your IP address:

- **[unpkg.com](https://unpkg.com/)** — serves KaTeX (math rendering), tablesort, and PapaParse (CSV parsing)
- **[Google Fonts](https://fonts.google.com/)** — serves the site's typeface
- **[Plotly CDN](https://cdn.plot.ly/)** — serves interactive chart libraries on some pages

No data is stored, tracked, or shared by these services on our behalf.
